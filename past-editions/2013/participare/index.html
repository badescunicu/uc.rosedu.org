Participare Upstream Challenge

Pentru a participa la concurs trebuie să:

  1. vă înscrieți pe lista de discuții
  2. identificați un bug
  3. rezolvați bug
  4. trimiteți bug pe lista de discuții

În continuare, pentru următoarele contribuții se vor urma pașii 2-

Citiți și regulamentul și lista de proiecte și contribuții acceptate. Consultați pagina de resurse pentru alte informații.

Concursul este sponsorizat de Intel Romania. Consultați regulamentul pentru detalii.
